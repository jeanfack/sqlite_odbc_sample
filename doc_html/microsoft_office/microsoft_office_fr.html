Comment configurer une source de donnée SQLite sur Microsoft Office Excel ?

  1. Lancer Microsoft Office Excel.
  2. Créer un nouveau classeur.
  3. Enregistrer ce classeur sous C:\sqlite_odbc_sample\scripts\microsoft_office\ex1.xlsx.



  4. Aller dans Data->Get Data->From Other Sources->From ODBC.



    1. Selectionner "ex1_sqlite3_32bits_sync_NORMAL" (Idem pour "ex1_sqlite3_64bits_sync_NORMAL").
    2. Cliquer sur "OK".



  5. Dans la fenêtre Navigator
    1. Selectionner la table "USER".
    2. Cliquer sur "Load".







  6. Pour exporter la connexion :
    1. Aller dans Data->Queries & Connections



    2. Cliquer droit sur la table "USER".
    3. Cliquer sur "Export Connection File...



    4. Enregistrer sous C:\sqlite_odbc_sample\scripts\microsoft_office\ex1_sqlite3_32bits_sync_NORMAL.odc
  7. Comment actualiser ?
    1. Lancer cmd.exe dans C:\sqlite_odbc_sample\scripts.
    2. Executer ex1.INSERT.Charlie.Chaplin.cmd.



    3. Actualiser le tableau croisé.



    4. Observer l'insertion du nouvelle enregistrement.



    5. Executer ex1.DELETE.Charlie.Chaplin.cmd.



    6. Actualiser et observer la suppression de l'enregistrement.
