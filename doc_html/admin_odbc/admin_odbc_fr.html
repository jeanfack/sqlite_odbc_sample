Comment configurer une source de donnée utilisateur sur Windows ?

  1. Configuration 32 bits :
    1. Executer C:\Windows\syswow64\odbcad32.exe.
    2. Sélectionner l'onglet "Sources de données utilisateur".
    3. Cliquer sur "Ajouter".



    4. Dans la fenêtre Créer une nouvelle source de données :
      1. Selectionner "SQLite3 ODBC Driver".
      2. Cliquer sur "Terminer".



    5. Dans la fenêtre SQLite3 ODBC DSN Configuration :



      1. Saisir dans Data Source Name: "ex1_sqlite3_32bits_sync_NORMAL".
      2. Parcourir dans Database Name: "C:\sqlite_odbc_sample\scripts\sqlite\db\ex1.db".
      3. Selectionner dans Sync Mode: "NORMAL".
      4. Cliquer sur "OK".



    6. Cliquer sur "OK".



  2. Configuration 64 bits :
    1. Executer C:\Windows\system32\odbcad32.exe.
    2. Sélectionner l'onglet "Sources de données utilisateur".
    3. Cliquer sur "Ajouter".



    4. Dans la fenêtre Créer une nouvelle source de données :
      1. Selectionner "SQLite3 ODBC Driver".
      2. Cliquer sur "Terminer".



    5. Dans la fenêtre SQLite3 ODBC DSN Configuration :



      1. Saisir dans Data Source Name: "ex1_sqlite3_64bits_sync_NORMAL".
      2. Parcourir dans Database Name: "C:\sqlite_odbc_sample\scripts\sqlite\db\ex1.db".
      3. Selectionner dans Sync Mode: "NORMAL".
      4. Cliquer sur "OK".



    6. Cliquer sur "OK".
