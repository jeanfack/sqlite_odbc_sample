Comment construire une base de donnée SQLite sur Windows 10 ?

  1. Lancer cmd.exe dans C:\sqlite_odbc_sample\scripts.

  2. Executer ex1.del.cmd.
    => Supprime le fichier ex1.db.



  3. Executer ex1.new.cmd.
    => Crée le fichier ex1.db qui contient une base de donnée vide.



  4. Executer ex1.CREATE.cmd.
    => Crée la table USER.



  5. Executer ex1.INSERT.cmd.
    => Insère 5 enregistrements dans la table USER.



  6. Executer ex1.SELECT.cmd.
    => Affiche la table USER.



  7. Executer ex1.INSERT.Charlie.Chaplin.cmd.
    => Insert un 6-ème enregistrement dans la table USER.



  8. Executer ex1.SELECT.cmd.
    => Affiche la table USER.



  9. Executer ex1.DELETE.Charlie.Chaplin.cmd.
    => Insert un 6-ème enregistrement dans la table USER.



  10. Executer ex1.SELECT.cmd.
    => Affiche la table USER.
