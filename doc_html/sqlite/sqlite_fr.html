Comment construire une base de donnée SQLite sur Windows 10 ?

  1. Archive SQLite_HowTo_sample.zip:
    1. Télécharger de https://github.com/jeanfack.
    2. Décompresser dans C:.
  2. Archive sqlite-tools-win32-x86-3320300.zip:
    1. Télécharger de https://www.sqlite.org/download.html.
    2. Décompresser dans C:\SQLite_HowTo_sample.
  3. Scripts .cmd:
    1. Lancer cmd.exe dans C:\SQLite_HowTo_sample\scripts.

    2. Executer ex1.all.cmd.



    3. Executer ex1.SELECT.cmd.
