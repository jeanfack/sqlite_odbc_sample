Comment installer le driver ODBC SQLite ?

  1. Coonfiguration 32 bits:
    1. sqlitodbc.exe:
      1. Télécharger sqlitodbc.exe de http://www.ch-werner.de/sqliteodbc/.
      2. Copier sqlitodbc.exe dans C:\SQLite_HowTo_sample\www.ch-werner.de_sqliteodbc.
      3. Executer C:\SQLite_HowTo_sample\www.ch-werner.de_sqliteodbc\sqlitodbc.exe. Cela va installer le driver.
    2. odbcad32.exe:
      1. Lancer C:\Windows\syswow64\odbcad32.exe.
      2. Sélectionner l'onglet "Pilotes ODBC".
      3. Vérifier que le pilote "SQLite3 ODBC Driver" est présent.
      4. Cliquer sur "OK".



  2. Coonfiguration 64 bits:
    1. sqlitodbc_w64.exe:
      1. Télécharger sqlitodbc_w64.exe de http://www.ch-werner.de/sqliteodbc/.
      2. Copier sqlitodbc_w64.exe dans C:\SQLite_HowTo_sample\www.ch-werner.de_sqliteodbc.
      3. Executer C:\SQLite_HowTo_sample\www.ch-werner.de_sqliteodbc\sqlitodbc_w64.exe. Cela va installer le driver.
    2. odbcad32.exe:
      1. Lancer C:\Windows\system32\odbcad32.exe.
      2. Sélectionner l'onglet "Pilotes ODBC".
      3. Vérifier que le pilote "SQLite3 ODBC Driver" est présent.
      4. Cliquer sur "OK".
