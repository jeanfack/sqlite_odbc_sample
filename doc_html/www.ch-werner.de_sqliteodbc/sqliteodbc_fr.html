Comment installer le driver ODBC SQLite sur Windows ?

  1. Coonfiguration 32 bits:
    1. Executer C:\SQLite_HowTo_sample\www.ch-werner.de_sqliteodbc\sqlitodbc.exe. Cela va installer le driver.
    2. Executer C:\Windows\syswow64\odbcad32.exe.
      1. Sélectionner l'onglet "Pilotes ODBC".
      2. Vérifier que le pilote "SQLite3 ODBC Driver" est présent.
      3. Cliquer sur "OK".



  2. Coonfiguration 64 bits:
    2. Executer C:\Windows\system32\odbcad32.exe.
      1. Sélectionner l'onglet "Pilotes ODBC".
      2. Vérifier que le pilote "SQLite3 ODBC Driver" est présent.
      3. Cliquer sur "OK".
