Comment construire une base de donnée SQLite sur Windows 10 ?

  1. Installer GitHub Desktop: https://desktop.github.com/.
  2. Aller dans File->Clone Repository...
    1. Saisir url: https://github.com/jeanfack/sqlite_odbc_sample.git
    2. Parcourir path: C:\sqlite_odbc_sample
    3. Cliquer sur "Clone".
